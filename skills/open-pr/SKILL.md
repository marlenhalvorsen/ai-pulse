# Open PR

## Pre-conditions
- `dotnet build` zero warnings + `dotnet test` all green
- /review passed
- Not on main

## PR body template
```
> 🤖 This PR was generated by Claude Code under human supervision.

## Summary
{what was built and why}

## Tests
{new tests and what they verify}

## Decisions
{non-obvious choices and why}

🤖 Generated with Claude Code
```

## Command
```
gh pr create --title "type: description" --body "..."
```

## Stop
**STOP COMPLETELY after opening PR.**
Do not write code. Do not merge. Wait for human "continue".
